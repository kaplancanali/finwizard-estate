# Torkam Property Management UI

Modern property management frontend for Torkam Holding.

## Brand

- Company: **Torkam**
- Legal: **Torkam Holding**
- Logo: `public/torkam-logo.svg`

## Getting Started

```bash
cd frontend
npm install
cp .env.example .env.local
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
